# Titanic Dataset - Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data through summary statistics and visualizations.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Generated summary statistics (mean, median, standard deviation, etc.)
- Created histograms for numerical features
- Visualized outliers using boxplots
- Explored feature relationships using pairplots
- Analyzed correlations with a heatmap
- Identified patterns, trends, and anomalies in the dataset
- Drew feature-level insights based on the visualizations

## Outcome
The analysis provides a better understanding of the dataset and highlights relationships between features, serving as a foundation for machine learning model development.
